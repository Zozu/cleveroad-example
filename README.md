# node-example


```
Usage:

1. Checkout project to your desktop
2. Create local MySQL database named 'test_db' on 3306 port
3. Execute test_db.sql in ./testproj/db
4. cd to ./testproj
5. Make 'npm install'
6. Run server by this command 'set DEBUG=testproj && node ./bin/www'
7. Now you can view project on http://localhost:3000
```
